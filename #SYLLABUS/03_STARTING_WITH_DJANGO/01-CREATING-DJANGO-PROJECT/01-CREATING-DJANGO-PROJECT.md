SETTING UP ENVIRONMENT AND CREATING NEW PROJECT

- Create Folder Ex. `'firstProject'`
- Open CMD (Commanda Prompt) then Type Following Commands

`1) python -m venv myenv  ( It Creates 'myenv' Folder )` </br>
`2) myenv\Scripts\activate` </br>
`3) pip install django` </br>
`4) django-admin startproject myproject  ( 'myproject' is a Project Name : It Display as a Folder )` </br>
`5) cd myproject` </br>
`6) code .` </br>
   
- When we type (` code . `) then VS Code will be Open.
- Then Open New Terminal and Select Command Prompt. 

  <br/> ![UI Screenshot](01_SELECT_COMMAND_PROMPT.png) <br/>
 
- Then Type : `python manage.py runserver`

  <br/> ![UI Screenshot](02_WRITE_COMMAND_TO_RUN_PROJECT.png) <br/>
