DYNAMIC URL ROUTING

- Go To 'urls.py' [ In All App ]
- Add Third Argument 'name'. [ name=' ' ]
- Give the name for this URL. ( Ex. `name='ele_home'` ) </br></br>

- Add this name like this </br>
