----- APP CREATION -----

- Open Terminal in Command Prompt
- Type: `python manage.py startapp ecom` ( Here 'ecom' is App Name )

</br> <img width="1920" height="1080" alt="01_CREATE_APP" src="https://github.com/user-attachments/assets/cbbc127c-9972-4857-bb96-59ac44211d5c" /></br>

- Then App will be Create.

</br><img width="1920" height="1080" alt="02_APP_CREATED" src="https://github.com/user-attachments/assets/f373fbee-741c-484c-99d5-9c79b3c6948f" /></br>


- Go To `' settings.py '` in `myproject` File.
- Add app name `ecom` in `INSTALLED_APPS` List.
